# Code of Conduct

โปรเจกต์นี้ยินดีต้อนรับทุกคนที่สนใจเรียนรู้และพัฒนาต่อยอดร่วมกัน

## หลักการ
- ให้เกียรติและสื่อสารอย่างสุภาพ
- เปิดรับความคิดเห็นและคำแนะนำ
- โฟกัสที่ข้อเท็จจริงและผลลัพธ์ ไม่โจมตีบุคคล

## สิ่งที่ไม่ยอมรับ
- การคุกคาม เหยียดหยาม หรือข้อความที่ทำให้ผู้อื่นไม่ปลอดภัย
- การเผยแพร่ข้อมูลส่วนบุคคลของผู้อื่นโดยไม่ได้รับอนุญาต

## การรายงานปัญหา
หากพบพฤติกรรมไม่เหมาะสม ให้เปิด Issue พร้อมรายละเอียดเหตุการณ์ (วันเวลา/บริบท) ผู้ดูแลจะพิจารณาดำเนินการตามความเหมาะสม
